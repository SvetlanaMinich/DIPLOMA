# План разработки системы AutoSTP

## Обзор проекта

AutoSTP - веб-платформа для автоматизации оформления курсовых и дипломных работ студентов БГУИР в соответствии со стандартами СТП 01-2024. Система принимает неоформленный текстовый документ и формирует полностью отформатированный документ, готовый к сдаче.

**Ключевые принципы:**

- Сквозная автоматизация без обязательного участия пользователя в разметке структуры
- Применение LLM для семантической сегментации
- Декларативное описание стандарта через конфигурационные шаблоны
- Уровневая модель доступа (базовый и премиум)

**Технологический стек:**

- **Backend**: FastAPI (Python), async/await
- **Frontend**: React, Slate.js/Draft.js для редактора
- **Database**: PostgreSQL с типом JSONB
- **Deployment**: Docker, Nginx
- **AI Integration**: OpenRouter API (LLaMA 3 8B и другие модели)
